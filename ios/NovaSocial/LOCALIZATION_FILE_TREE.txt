🚀 使用示例
═══════════════════════════════════════════════════════════════════

基本文本:
    Text(L10n.Common.cancel)

参数化字符串:
    Text(L10n.Post.likesCount(42))
    // Output: "42 个赞" / "42 likes"

日期格式化:
    Text(date.fullDateString)
    // zh-Hans: "2024年10月19日"
    // en: "October 19, 2024"

数字格式化:
    Text(number.compactString)
    // "1.2M"

语言切换:
    localizationManager.setLanguage(.english)
    // 所有 View 自动刷新
