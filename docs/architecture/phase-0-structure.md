## 下一阶段集成点

Phase 1 将在以下位置添加实现:

1. **src/services/**
   - `auth_service.rs` - 认证业务逻辑
   - `email_service.rs` - 邮件发送服务
   - `token_service.rs` - JWT 令牌管理

2. **src/utils/**
   - `password.rs` - Argon2 密码哈希
   - `token_gen.rs` - 安全令牌生成
   - `hash.rs` - SHA256 哈希工具

3. **src/handlers/auth.rs**
   - 实现 `register()` 处理器
   - 实现 `verify_email()` 处理器

4. **tests/**
   - 单元测试
   - 集成测试
   - API 测试
